# Koa Proxies
一定要全局安装(koa1.2和koa2都己经支持)，
koa1.2 生成一个test项目,切到test目录并下载依赖 
koa2 生成一个test项目,切到test目录并下载依赖 
```bash
npm install koa-generator -g 
koa test && cd test && npm install
koa2 test && cd test && npm install
```
